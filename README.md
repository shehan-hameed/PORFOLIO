# Shehan Hameed — Personal Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript as part of the **IN1601 Multimedia Technologies and Web Design** assignment at the University of Moratuwa.

---

## 🔗 Live Demo

[View Portfolio](https://shehan-hameed.github.io/PORFOLIO/)

---

## 📄 Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page — Hero, About, Skills, Education, Achievements, Leadership |
| `blog.html` | Blog page — Link to published Medium article |
| `contact.html` | Contact page — Form with JavaScript validation |

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Dark and modern UI with purple accent theme
- Smooth scroll single-page navigation on Home
- JavaScript form validation on Contact page (no backend required)
- Animated skill tags and hover effects
- Timeline layout for Education section
- Achievements and Leadership sections
- Linked Medium blog post

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Media Queries)
- JavaScript (DOM manipulation, form validation)
- Google Fonts (Syne, DM Sans)

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/shehan-hameed/PORFOLIO.git
   ```
2. Open `index.html` in your browser — no server or installation needed.
